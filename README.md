10. 跳ap再跳wowza因為分片請求過大不行，上傳到ap再上傳到wowza不行因為先前分片的請求已經沒了，直接打wowza伺服器不行會被cors擋掉並且因為WOWZA理論上是不能對外公開也不能掠過，網芳共用資料夾不行，資安不會過，
